# Шпаргалка по терминалу Linux (Ubuntu)

## Навигация по файловой системе

| Команда             | Описание |
|---------------------|----------|
| `pwd`               | Показать текущую директорию |
| `ls`                | Показать список файлов в текущей директории |
| `ls -l`             | Подробный список: права, владелец, размер, дата |
| `ls -a`             | Показать скрытые файлы (начинающиеся с `.`) |
| `cd путь`           | Перейти в указанную директорию |
| `cd ..`             | Подняться на уровень выше |
| `cd -`              | Перейти в предыдущую директорию |
| `cd ~` или `cd`     | Перейти в домашнюю директорию |
| `tree`              | Древовидное отображение (можно установить через `sudo apt install tree`) |


## Работа с файлами и папками

| Команда                      | Описание |
|------------------------------|----------|
| `touch файл`                 | Создать пустой файл |
| `mkdir папка`                | Создать папку |
| `mkdir -p путь/в/папку`      | Создать вложенные папки |
| `cp источник цель`           | Копировать файл |
| `cp -r папка1 папка2`        | Копировать папку и содержимое |
| `mv файл1 файл2`             | Переместить или переименовать файл |
| `rm файл`                    | Удалить файл |
| `rm -r папка`                | Удалить папку и содержимое |
| `rm -f файл`                 | Удалить без подтверждения |
| `stat файл`                  | Информация о файле: размер, дата, права |
| `file файл`                  | Определить тип файла |


## Просмотр и редактирование файлов

| Команда            | Описание |
|--------------------|----------|
| `cat файл`         | Показать содержимое файла |
| `less файл`        | Просмотр файла с прокруткой |
| `head файл`        | Показать первые 10 строк |
| `tail файл`        | Показать последние 10 строк |
| `tail -f файл`     | Следить за обновлением файла в реальном времени (например, лог) |
| `nano файл`        | Удобный встроенный редактор (Ctrl+O — сохранить, Ctrl+X — выйти) |
| `vim файл`         | Продвинутый редактор (нужно изучать отдельно) |


## Поиск

| Команда                                     | Описание |
|---------------------------------------------|----------|
| `find /путь -name "имя"`                    | Поиск по имени |
| `find . -type f -name "*.log"`              | Найти все `.log` в текущей папке |
| `grep "строка" файл`                        | Поиск строки в файле |
| `grep -r "строка" папка/`                   | Рекурсивный поиск по папке |
| `locate имя`                                | Быстрый поиск по базе (требует `updatedb`) |
| `which команда`                             | Путь к исполняемому файлу |
| `whereis команда`                           | Расширенный поиск исполняемых, исходников, man-файлов |


## Пакеты и обновления (APT)

| Команда                              | Описание |
|--------------------------------------|----------|
| `sudo apt update`                   | Обновить список пакетов |
| `sudo apt upgrade`                  | Обновить установленные пакеты |
| `sudo apt install имя`             | Установить пакет |
| `sudo apt remove имя`              | Удалить пакет |
| `sudo apt purge имя`               | Удалить с конфигами |
| `sudo apt autoremove`              | Удалить неиспользуемые зависимости |
| `apt list --installed`             | Список установленных пакетов |
| `dpkg -l`                           | То же, но детальнее |
| `dpkg -S /путь/к/файлу`             | Какой пакет установил этот файл |


## Работа с архивами

## Cron

`crontab -l` - посмотреть список задач

`crontab -e` - отредактировать задачи

`crontab -r` - удалить все задачи

### Пример записи

`*/15 3,6 8-12 * * echo 'Cron' >> /home/ubuntu22/Desktop/tmp.txt` 

Порядок: минуты, часы, день месяца, месяц, день недели (0-6 вс-сб)  

Через слэш каждые, через запятую конкретные, через дефис интервал  

**Что значит эта запись:** каждые 15 минут в 3 и 6 часов с 8 по 12 число месяца записывать слово Cron в файл tmp.txt  

Можно проверять в гугле через crontab validator  


## Советы

- Используй `man команда` для получения справки по любой команде.
- Установи `htop`, `ncdu`, `bat`, `ripgrep`, `zoxide` — они делают терминал удобнее.
- Используй `.bashrc` или `.zshrc` для кастомизации среды: alias, PATH, функции.
