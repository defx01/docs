# Шпаргалка по alt кодам Windows

Чтобы напечатать символ, необходимо зажать alt + указанные цифры на num клавиатуре, затем отпустить alt

## Типография

7 = •  
21 = §  
22 = ▬  
0150 = –  
0151 = —  
0171 = «  
0187 = »  
96 = `  
35 = #  

## Копирайты

0153 = ™  
0169 = ©  
0174 = ®  

## Стрелки

16 = ►  
17 = ◄  
30 = ▲  
31 = ▼  
24 = ↑  
25 = ↓  
26 = →  
27 = ←  

## Карточные масти

5 = ♣  
6 = ♠
