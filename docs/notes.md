- Комментарии
- Все типы данных и определение типа
- Интерполяция
- Операторы
- Управляющие конструкции
- Циклы
- Функции
- Массивы, Объекты
- ООП
- Обработка ошибок
- Дата и время
- Регулярные выражения
- Хранение данных
- Асинхронный код
- Современные возможности

- Особенности языка
